# Examples — Product Value Governance

Worked walkthroughs of the pack in use.

- [`build-walkthrough.md`](./build-walkthrough.md) — full orchestration: lever map →
  complexity audit → gates → Build Now verdict on the contract.
- [`park-walkthrough.md`](./park-walkthrough.md) — a plausible idea with no capacity,
  parked with an explicit resumption trigger.

The AletheIA repo carries the contract-level decision records for **Build**, **Test**,
**Kill**, and **Sunset** under
[`examples/feature-governance/`](../../../../aletheia/examples/feature-governance/). These
walkthroughs show the *skill composition* that produces such records.
