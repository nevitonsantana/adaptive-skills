# Bounded Slice Example

## Oversized request
Ship the whole operational dashboard refresh with tracking, copy cleanup, and alerting changes.

## Current slice
Clarify one dashboard approval summary so the state is easier to scan and validate.

## In this slice
- one approval summary surface
- copy adjustment
- minimal proof for the changed state

## Not in this slice
- dashboard redesign
- new metrics taxonomy
- alerting behavior changes

## Required dependencies
- existing summary component
- current validation path

## Stop condition
The chosen summary surface is clearer and the proof for that local change passes.

## Next slice trigger
Another scan-heavy dashboard surface shows the same problem pattern.
