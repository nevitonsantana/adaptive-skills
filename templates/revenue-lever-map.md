# Template — Revenue Lever Map

Output template for `revenue-lever-mapping`. One primary lever; metric and proxy attached.

| Field | Value |
| --- | --- |
| **Opportunity** | One-line statement. |
| **Primary lever** | Exactly one: acquisition / activation / retention / expansion / efficiency / margin / strategic_defense. |
| **Primary rationale** | The mechanism, not a feeling. |
| **Secondary levers** | Subordinate levers, if any. |
| **Metric** | Name + direction (up/down) + baseline if known. |
| **Proxy** | Faster early signal, and when it can mislead. |
| **Risks** | What would make this the wrong lever story. |
| **Uncertainty** | What is not yet known. |
| **Mode** | generic / knowledge_aware (+ `strategic_framework_ref`). |
