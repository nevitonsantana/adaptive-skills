# Example — clarify intent before changing the Mission Control

## Request

“Add success rates for skills and agents to the Observatory.”

## Clarification record

- **Human-owned outcome:** understand which skills and agents contribute to repeatable successful work without turning correlation into authority.
- **Proposed solution:** show success-rate metrics in the Resource Observatory.
- **Constraints:** read-only projection; source-backed records only; no automatic ranking or routing.
- **Non-goal:** deciding which skill or agent must be used next.
- **Observable success:** a reviewer can trace a displayed comparison to comparable accepted outcomes and source references.
- **Observable failure:** the interface shows percentages derived from incomparable tasks or missing acceptance evidence.
- **Evidence hole:** fewer than five comparable reviewed records exist in a shared comparison group.
- **Availability:** comparable success rate is 'unavailable'.
- **Guessing risk:** high — displaying a percentage now would invent confidence.
- **Advisory posture:** clarify and defer the metric; continue collecting source-backed execution observations.
- **Human confirmation:** pending.

The record can support an AletheIA Intent-to-Evidence Work Slice. It does not create, approve, or close that slice.
