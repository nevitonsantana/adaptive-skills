Adaptive Skills is a portable library of micro-skills for disciplined AI-assisted work.

Use this documentation site to understand the model, install or consume skills, choose the right skill, and review the evidence behind the official docs publication path.

## How to read the system

<Steps>
  <Step title="Start with the model">
    Read the overview first. It explains Adaptive Skills as portable micro-capabilities, not a runtime engine.
  </Step>
  <Step title="Choose a path">
    Use the installation and catalog pages when you want to consume skills; use ADRs when you need the design rationale.
  </Step>
  <Step title="Check the evidence">
    Publication, link-readiness, and Blume-shell pages show what is official, what was validated, and what remains manual.
  </Step>
</Steps>

  </AccordionItem>
  <AccordionItem title="Does this site change skill behavior?">
